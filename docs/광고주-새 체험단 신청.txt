광고주의 체함단 관리 페이지에서 [새 체험단 신청]를 클릭하면, 
- Step.1: 기본 정보 출력
    - 광고주 정보 출력
    - 체험단 약관 동의
    - 매장 유형 선택
        - 화면 구성: 두 개의 큰 카드 버튼 (이미지 + 텍스트)
        - A 버튼 (오픈 예정):
            - 카피: "곧 오픈하시나요? 오픈빨 90일, 빈손으로 놓치지 마세요."
            - 설명: 신규 매장 전용 오픈런 패키지 (사전 촬영 + 리뷰 세팅)
        - B 버튼 (매장 운영 중):
            - 카피: "열심히 하는데 순위가 그대로인가요? 진단부터 다시 해드립니다."
            - 설명: 순위 상승 솔루션 & 리브랜딩 패키지
- Step.2: 요금제 선택(Step.1에서 선택한 유형에 따라 첫 번째 요금제의 내용이 다름)
    - Step.1에서 A 버튼을 선택한 경우:
        - 요금제 1: Basic 29만원
            - 오픈 준비 팩
            - [안전빵] "리뷰 0개인 민망한 상태만 피하자." (사진/기본리뷰 확보)
        - 요금제 2: Pro 49만원
            - 그랜드 오픈 팩
            - [줄세우기] "오픈 첫 주에 리뷰 50개 깔아서 기선 제압하자."
        - 요금제 3: VIP 79만원
            - 런칭 컨설팅 팩
            - [설계] "첫 단추부터 전문가가 끼워준다. SEO/키워드 완벽 세팅."
    - Step.1에서 B 버튼을 선택한 경우:
        - 요금제 1: ① 29만원 (실속형)
            - 자료 수집 & 기본 리뷰
            - 1팀 (5명) / Lv1 팀장
            - 25건 + 원본 사진 확보

        - 요금제 2: ② 49만원 A (기술형)
            - 리뷰 + 순위 부스팅
            - 1팀 (5명) / Lv1 팀장
            - 25건
            - 유입 3,000 + 저장 100

        - 요금제 3: ③ 49만원 B (물량형)
            - 리뷰 폭격 (물량 2배)
            - 2팀 (10명) / Lv1 팀장
            - 50건 (도배 효과)

        - 요금제 4: ④ 79만원 (VIP형)
            - 지역 1등 만들기 (Total)
            - 2팀 (10명) / Lv2 프로
            - 50건 (고퀄리티 보장)
            - 유입 5,000 + 저장 300

- Step.3: 결제 정보 입력 (결제 방법(카드/무통장 입금), 결제 금액, 결제 일시, 세금계산서 발행 정보)
- Step.4: 결제 완료 (결제 완료 화면)
- Step.5: 추가 정보 입력(Step.1에서 선택한 유형에 따라 추가 정보 입력)
    - 이전에 체험단을 운영한 적이 있고, 사전에 입력된 정보가 있다면 기존 정보를 출력하고 수정할 수 있도록 한다.
    - Step.1에서 A 버튼을 선택한 경우:
        - 상호명 (가칭)
        - 오픈 예정일 (D-Day)
        - 체험단 촬영 가능 시작일
        - 매장 위치 (주소)
        - 촬영용 대표 메뉴
        - 지오래빗 제공 혜택 (5만/10만 등)
        - 담당자 휴대폰 번호
    - Step.1에서 B 버튼을 선택한 경우:
        - 상호명 (네이버 검색 연동 권장)
        - 현재 네이버 플레이스 순위 고민 (유입/순위/리뷰수 중 택1)
        - 월 방문 가능 팀 수
        - 매장 위치
        - 대표 메뉴
        - 지오래빗 제공 혜택
        - 담당자 휴대폰 번호
- Step.6: 광고주 회원 가입 완료 (광고주 회원 가입 완료 화면)
    - 메인 카피: "접수가 완료되었습니다! 담당 매니저가 24시간 내에 배정됩니다."
    - 안내 사항:
        - "입력하신 [010-XXXX-XXXX] 번호로 담당자가 해피콜을 드립니다."
        - "나머지 상세한 매장 정보(영업시간, 브랜드 철학 등)는 매니저와 통화하며 채워드릴 예정이니 걱정 마세요."
        - [Action] 가입완료 이메일 발송과 카카오톡 알림톡 발송
        - 해당 체험단 신청은 "신청중" 상태가 된다.
